# Anifa Homeschooling Website

Modern React website for Anifa Homeschooling built with Vite, TypeScript, and Tailwind CSS.

## Features

- ✅ React 18 with TypeScript
- ✅ Vite for fast development and optimized builds
- ✅ Tailwind CSS for styling
- ✅ Mobile-first responsive design
- ✅ SEO optimized with meta tags and structured data
- ✅ Horizontal scrolling gallery
- ✅ Netlify deployment ready

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment

This project is ready to deploy on Netlify. Simply connect your repository and Netlify will automatically build and deploy your site.

### Environment

- Node.js 18+
- npm or yarn
